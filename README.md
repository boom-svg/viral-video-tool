# 爆款视频内容复刻辅助工具

一个基于AI的短视频脚本分析和生成工具，帮助内容创作者快速复刻爆款视频的逻辑结构。

## 功能特点

- 📤 多文件上传：支持同时上传多个视频/音频文件
- 🔍 智能分析：提取爆款视频的脚本结构和公式
- 💬 对话式生成：像ChatGPT一样多轮对话修改脚本
- 🎬 素材推荐：AI推荐匹配的图片和视频素材
- 📦 项目管理：支持创建多个项目，分别管理

## 快速开始

### 1. 安装依赖

```bash
pip install streamlit openai requests python-multipart
```

### 2. 配置API密钥

在应用侧边栏输入DMXAPI密钥，或设置环境变量：

```bash
export DMXAPI_KEY="your-api-key"
```

### 3. 运行应用

```bash
streamlit run app.py
```

## 项目结构

```
.
├── app.py              # 主应用文件
├── prompts.py          # AI提示词模板
├── assets.py           # 素材搜索和生成模块
├── requirements.txt    # 依赖列表
└── README.md          # 项目说明
```

## 使用流程

1. **创建项目**：在侧边栏输入项目名称并点击新建
2. **上传素材**：上传参考视频/音频文件
3. **分析结构**：AI自动分析爆款公式
4. **生成脚本**：输入新主题，AI生成脚本
5. **对话修改**：多轮对话优化脚本内容
6. **素材匹配**：AI推荐匹配的图片/视频素材
7. **导出项目**：导出完整的项目文件

## 技术栈

- **前端框架**：Streamlit
- **AI模型**：DMXAPI (GPT-5, Whisper, DALL-E等)
- **编程语言**：Python 3.8+

## 注意事项

- 请确保API密钥有足够的调用额度
- 音视频文件大小建议不超过100MB
- 素材搜索功能需要网络连接
